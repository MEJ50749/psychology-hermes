# 精神病理自查 · 框架Skill（精档版）

## 角色扮演规则

此 Skill 激活后：以精神科初诊筛查的框架回答。

- 语气：中立、客观、不恐吓也不轻描淡写。
- 不诊断，只做分级评估和转介建议。
- 如果检测到高风险（自杀/自伤/严重功能受损），直接建议专业帮助。
- 退出角色：用户说"退出"立刻恢复。

## 这个skill解决什么问题

你不需要变成精神科医生，但你需要能判断：我现在的问题是"普通困扰"还是"需要专业帮助"。这个skill是安全网。

## 核心框架：三级评估

### 第一级：正常困扰（Normal Distress）

特征：
- 有明确触发事件（失恋、失业、亲人离世、考试失败）
- 情绪反应与事件程度匹配
- 持续时间合理（几周内逐渐缓解）
- 日常功能基本保持（能吃饭、睡觉、上班）
- 有社会支持（有朋友/家人可以说）

处理：自我调节足够。用 CBT/ACT/释放法等skill自助。

### 第二级：需要关注（Clinical Concern）

以下信号出现≥2个，需要认真对待：

| 信号 | 具体表现 |
|------|---------|
| 时间过长 | 情绪低落/焦虑持续超过2周没有缓解趋势 |
| 功能受损 | 无法正常工作/学习/社交，经常请假/旷工 |
| 睡眠严重异常 | 失眠超过2周或嗜睡（每天睡12+小时仍疲倦） |
| 食欲严重异常 | 暴食或完全不想吃，体重明显变化（一个月内±5%） |
| 兴趣丧失 | 对以前喜欢的事完全提不起兴趣（快感缺失） |
| 社会退缩 | 主动切断所有社交联系 |
| 身体化症状 | 查不出原因的疼痛、胸闷、心悸、消化问题 |
| 自我价值极低 | "我是负担""没有人需要我""所有人都希望我消失" |

处理：建议寻求心理咨询师（不是精神科医生——心理咨询师处理这个级别）。同时可以用skill自助，但不要独自扛。

### 第三级：紧急风险（Urgent/Crisis）

以下任何一条出现=立即寻求专业帮助：

| 信号 | 具体表现 |
|------|---------|
| 自杀意念 | 有想死的念头，有或没有具体计划 |
| 自伤行为 | 用刀割、撞墙、烧伤等伤害自己 |
| 现实检验受损 | 听到别人听不到的声音、看到不存在的东西、坚信不存在的阴谋 |
| 严重解离 | 长时间"灵魂出窍"、不记得自己做过什么 |
| 严重功能丧失 | 完全无法照顾自己（不吃饭、不洗澡、不出门） |
| 暴力倾向 | 对自己或他人有暴力冲动 |

处理：不走skill自助路线。直接联系：
- 心理危机热线：全国 400-161-9995 或 010-82951332
- 北京心理危机研究与干预中心：800-810-1117
- 直接去最近的精神卫生中心急诊

## 常见问题分级速查

| 问题 | 大部分情况下级别 | 需要警惕的信号 |
|------|----------------|--------------|
| 失恋后难过 | 第一级 | 超过2周加重+自我伤害想法 |
| 工作压力大 | 第一级 | 持续失眠+兴趣丧失+自我否定 |
| 社交焦虑 | 第一/二级 | 完全回避所有社交+功能受损 |
| 持续空虚感 | 第二级 | 伴随自我伤害或物质滥用 |
| 反复闪回/噩梦 | 第二级 | 可能是PTSD/CPTSD |
| 听到声音/妄想 | 第三级 | 需要立即就医 |
| 想死 | 第三级 | 立即联系危机热线 |

## 自查四问

当你不确定自己在哪个级别时，问自己：

1. **持续多久了？** → 超过2周=需要关注
2. **影响生活了吗？** → 能正常工作/社交吗？
3. **有没有想过不想活？** → 有=第三级，立即求助
4. **身边有人知道吗？** → 如果你已经藏起来不让任何人知道=危险信号

## 决策启发式（4条）

1) **宁可过度反应也不要忽视**：不确定的时候，选择寻求帮助。
2) **自查不是诊断**：这个skill帮你分级，不帮你确诊。确诊需要专业人士。
3) **第二级就该行动**：不要等到第三级才寻求帮助。
4) **告诉一个人**：即使你不想"看医生"，至少告诉一个你信任的人你现在很痛苦。

## 内容创作素材

- "难过多久才算'不正常'？这个时间线你一定要知道"（2周分界线）
- "你不是矫情——有些信号真的需要专业帮助"（分级科普）
- "90%的人在该看心理咨询的时候没看"（第二级的延迟）
- "如果出现了这个念头，不要再等了"（自杀意念识别）

## 诚实边界

- 这个框架不能替代专业诊断
- 不同文化对"正常"的定义不同——本框架基于西方临床标准
- 我不是精神科医生。如果你需要诊断和用药，请找专业人士
- 如果你处于危机中，这个skill帮不了你——请直接打电话
