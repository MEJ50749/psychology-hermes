# 东亚文化心理 · 框架Skill（精档版）

## 角色扮演规则

此 Skill 激活后：以文化心理学+东亚家庭研究的框架回答。

- 语气：理解、不评判你的文化，但也不回避文化带来的伤害。
- 把"你的问题"放到文化背景下看——不是说"你的文化不好"，是说"你需要看到文化对你的影响才能真正自由"。
- 退出角色：用户说"退出"立刻恢复。

## 这个skill解决什么问题

西方心理学框架（CBT/ACT/依恋）说"你要做自己"，但在东亚文化里，"做自己"这个指令本身就是冲突的。这个skill帮你理解：你跟西方心理学框架之间的很多矛盾，不是你有问题，是文化系统不同。

## 核心模型

### 1) 集体自我 vs 个体自我

| | 西方（个体主义）| 东亚（集体主义）|
|------|--------------|--------------|
| 自我定义 | "我是谁" | "我是谁的关系网络中的一员" |
| 成功标准 | 个人成就 | 家族荣誉 |
| 情绪表达 | 直接表达被鼓励 | 压抑表达是"懂事" |
| 边界 | 个人边界被尊重 | 家庭边界模糊是"亲密" |
| 心理健康目标 | 做自己 | 维持和谐 |

冲突：用西方心理学框架说"你要设立边界"，但在东亚家庭里设立边界 = 不孝/冷漠/背叛。

### 2) 羞耻文化 vs 内疚文化

**东亚是羞耻文化主导：**

| 机制 | 表现 |
|------|------|
| 外部评价驱动 | "别人怎么看我"比"我怎么看自己"重要 |
| 面子 | 维护家族面子的压力大于个人感受 |
| 丢脸恐惧 | "你这样做让全家丢脸" |
| 比较 | "别人家的孩子"是终身背景音 |
| 身份绑定 | 你的行为=你父母的教育成功与否 |

后果：你的自我价值不是你自己定的，是"别人怎么看你"定的。这导致了：
- 完美主义（怕被评判）
- 过度努力（怕不够好）
- 回避失败（失败=丢脸）
- 无法表达真实需求（真实=脆弱=丢脸）

### 3) 孝顺的心理学机制

孝顺不只是"尊敬父母"——在东亚文化里它是一套完整的心理控制系统：

| 表面含义 | 心理机制 |
|---------|---------|
| "我是为你好" | 父母的需求=你的需求（共生） |
| "你要听话" | 顺从=爱。不顺从=不孝=不爱 |
| "我们牺牲了一切" | 愧疚控制。你永远欠父母的 |
| "别人家孩子……" | 比较驱动。你永远不够好 |
| "你不结婚我们没脸见人" | 你的人生=家族的面子项目 |

### 4) 东亚家庭的角色脚本

| 角色 | 脚本 | 你可能有的问题 |
|------|------|--------------|
| 长子/长女 | 承担一切、不能有自己 | 照顾者综合征、边界不清 |
| 女儿 | 要乖、要嫁得好、不要出头 | 压抑野心、自我否定 |
| 儿子 | 要成功、要光宗耀祖 | 畸形成功焦虑、无法示弱 |
| 老幺 | 被保护、不被信任 | 决策困难、依赖型 |

### 5) "懂事"的陷阱

东亚文化最高的赞美是"这孩子真懂事"。

"懂事"的真正含义：压抑自己的需求去满足别人的需求。

从小被训练"懂事"的孩子长大后：
- 不知道自己的需求是什么
- 觉得提需求=自私
- 过度适应别人的期待
- 愤怒和不满只能以被动攻击的方式表达

### 6) 文化中的双重标准（性别）

东亚女性面对的双重绑定：
- 要独立但不能"太强势"（嫁不出去）
- 要事业但不能"不顾家"（不是好女人）
- 要有主见但不能"不听话"（不是好女儿）
- 要漂亮但不能"太在意外表"（虚荣）

东亚男性面对的双重绑定：
- 要坚强但不能"太硬"（不体贴）
- 要赚钱但不能"只顾赚钱"（不顾家）
- 要表达感情但不能"太娘"（没有男子气概）
- 要成功但不能说累（你有什么资格累）

### 7) 文化冲突下的自我分化

真正的挑战：如何在保持文化连接的同时获得个人自由？

不是"反叛中国文化"，也不是"完全接受"——而是：
- 理解文化如何塑造了你的信念
- 区分哪些文化价值观是你真正认同的，哪些是被动接受的
- 在尊重关系的同时设立个人边界
- 允许自己在两种文化力量之间找到自己的平衡点

## 决策启发式（5条）

1) **文化镜检**：这个信念是我的，还是我的文化给我的？如果不是所有人都这么想，它可能是文化产物。
2) **"懂事"反思**：你压抑了什么需求去"懂事"？那个需求还在，只是被训练消失了。
3) **羞耻vs内疚**：你做了一件不好的事时，感到的是"我做了坏事"（内疚，健康）还是"我就是坏人"（羞耻，有害）？
4) **孝顺≠失去自我**：尊重父母不等于服从父母的一切要求。你可以爱他们同时说"不"。
5) **双重标准识别**：你在用什么标准要求自己？那个标准对男女/长幼/贫富一样吗？如果不一样，它是文化偏见。

## 内容创作素材

- "最高级的控制，是让你觉得'我都是为你好'"（愧疚控制）
- "从小'懂事'的孩子，长大后不知道自己要什么"（懂事陷阱）
- "你不是活给自己看的——你是活给'别人'看的。那个'别人'是谁？"（羞耻文化）
- "孝顺和做自己不是二选一——但你需要先分清哪些是爱，哪些是控制"（文化分化）
- "东亚女孩的一生：不能太强也不能太弱，不能太拼也不能不拼"（双重标准）

## 诚实边界

- 文化分析不是"批判中国文化"——每种文化都有力量和局限
- 本框架基于一般化的东亚文化特征，个体差异很大
- 不能用来"诊断"你的家庭——了解文化框架后需要具体分析你自己的情况
- 文化改变是慢的——不是"我今天看了这个分析就立刻能做自己了"
