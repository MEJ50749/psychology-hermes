# 🧠 Psychology Skills for Hermes Agent — 垂直心理学工具箱

12个心理学框架的精档蒸馏 Skill，适用于 [Hermes Agent](https://github.com/alchaincyf/hermes-agent)。

覆盖从自我觉察到行为改变的完整链条：**认知 → 情绪 → 关系 → 身体 → 文化 → 行动**。

---

## 核心治疗框架（6个）

| Skill | 创始人 | 核心 | 一句话 |
|-------|--------|------|--------|
| [cbt-beck](cbt-beck/) | Aaron Beck | 认知行为疗法 | 你的情绪不是事件决定的，是你对事件的解读决定的 |
| [act-hayes](act-hayes/) | Steven Hayes | 接纳承诺疗法 | 不消除痛苦，带着痛苦朝价值方向行动 |
| [frankl-logotherapy](frankl-logotherapy/) | Viktor Frankl | 意义疗法 | 人可以在任何处境中找到意义 |
| [jung-shadow](jung-shadow/) | Carl Jung | 分析心理学 | 你不愿意面对的那部分自己，才是最大的力量来源 |
| [attachment-theory](attachment-theory/) | Bowlby & Ainsworth | 依恋理论 | 你的亲密关系模式在童年就写好了，但可以改写 |
| [body-keeps-score](body-keeps-score/) | Bessel van der Kolk | 创伤心理学 | 创伤储在身体里，不只是心理问题 |

## 系统框架（6个）

| Skill | 核心 | 一句话 |
|-------|------|--------|
| [family-systems-bowen](family-systems-bowen/) | 家庭系统理论 | 你不只是你自己，你是三代人的情绪产物 |
| [motivational-interviewing](motivational-interviewing/) | 动机式访谈 | 改变不是被强迫的，是被引导出来的 |
| [case-conceptualization](case-conceptualization/) | 个案概念化 | 把一个人的问题画成地图，而不是贴标签 |
| [psychopathology-selfcheck](psychopathology-selfcheck/) | 精神病理自查 | 你现在的情绪状态在什么级别，需不需要专业帮助 |
| [behavioral-activation](behavioral-activation/) | 行为激活+习惯系统 | 不是想好了再做，是做了才能想好 |
| [east-asian-psychology](east-asian-psychology/) | 东亚文化心理 | 你的痛苦不只是你的，是你的文化给你的 |

## 推荐学习路径

### 自我拯救路线
1. `psychopathology-selfcheck` — 先评估自己的状态级别
2. `cbt-beck` — 识别扭曲思维
3. `act-hayes` — 学会带着痛苦行动
4. `body-keeps-score` — 检查有没有未处理的创伤
5. `behavioral-activation` — 从最小行动开始改变

### 关系问题路线
1. `attachment-theory` — 分析你的依恋类型
2. `family-systems-bowen` — 看三代人的关系模式
3. `east-asian-psychology` — 理解文化对你的影响
4. `jung-shadow` — 直面你在关系中投射的阴影

### 内容创作路线
每个skill都包含"内容创作素材"板块，可直接用于自媒体文案。

## 每个 Skill 包含

- 核心理论与背景
- 核心模型（附案例）
- 实操方法与练习
- 决策启发式
- 内容创作素材
- 诚实边界（不适合/不确定的领域）
- 快速测试
